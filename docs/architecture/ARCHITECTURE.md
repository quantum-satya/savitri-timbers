# STDP Architecture

This directory contains the long-term architectural documents for the
Savitri Timbers Digital Platform.

These documents define the website structure, user experience,
content strategy, implementation sequence, and future scalability.

Documents

1. Website Information Architecture
   Complete sitemap and page hierarchy.

2. User Journey Blueprint
   Visitor personas and conversion paths.

3. Content & SEO Strategy
   Content pillars, keyword clusters and SEO architecture.

4. Implementation Roadmap
   Recommended development sequence.

These documents are considered the source of truth for all future
implementation work.

Any major UX, navigation or content decision should first update
the relevant architecture document before implementation.
