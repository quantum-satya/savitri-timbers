# 1. Project Overview

This repository contains the source code and documentation for the official website of Savitri Timbers Pvt. Ltd.

The website represents the company's brand, products, services, and business capabilities in premium timber sourcing and wooden flooring manufacturing.

The project is intentionally built using lightweight web technologies to provide excellent performance, security, maintainability, and long-term scalability.

The website serves as a digital platform for brand building, customer engagement, and lead generation.

---

# 2. Business Overview

Savitri Timbers Pvt. Ltd. sources premium timber from trusted international suppliers and transforms it into high-quality wood products for the Indian market.

The company combines:

- Global timber sourcing
- Responsible procurement
- Precision manufacturing
- Professional customer service

to deliver premium timber solutions.

## Business Model

Savitri Timbers imports **timber (raw material)** and manufactures premium wooden flooring in India.

The company **does not** position itself as an importer of finished wooden flooring.

Our value proposition is:

> Source Premium Timber → Manufacture Premium Wooden Flooring → Deliver Quality Solutions

# 11. Development Principles

Every change to the website should follow these principles.

- Maintain a premium visual identity.
- Prefer simplicity over unnecessary complexity.
- Reuse existing CSS wherever practical.
- Avoid introducing frameworks unless absolutely necessary.
- Preserve existing functionality unless explicitly approved.
- Keep pages fast and responsive.
- Write clean, maintainable, production-ready code.
- Use semantic HTML and accessible markup.
- Optimize for desktop, tablet, and mobile devices.

# 15. Project Philosophy

This website is intended to grow steadily with the business.

Every enhancement should support one or more of the following objectives:

- Strengthen the Savitri Timbers brand.
- Build customer trust.
- Improve user experience.
- Generate qualified business enquiries.
- Showcase current business capabilities.
- Maintain a premium and professional online presence.

The project favors long-term maintainability over short-term shortcuts and prioritizes clarity, performance, and quality in every release.
